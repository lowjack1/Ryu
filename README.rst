----------------


:Tornado: Async Python Web Library + Web Server
:PostgreSQL: Relational Database System

Requirements
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. python 3.6+
2. tornado
3. aiohttp
4. asyncpg
5. python-dateutil

Features
---------

* Shows who joined and who left
* Shows list of users online
* Shows who is typing
* Join/Leave chat room features
